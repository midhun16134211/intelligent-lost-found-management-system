# Intelligent Lost and Found Management System (ILFMS)

A web-based platform built using **Python Django and SQLite** to help users report, track, and recover lost items efficiently.

This system provides a centralized environment where users can report lost or found belongings, search for matching items, communicate with other users, and submit ownership claims.

---

## 🚀 Features

- Report Lost Items
- Report Found Items
- Image Upload Support
- AI Duplicate Image Detection
- Automated Item Matching
- Claim Verification System
- Real-time Notifications
- Direct Messaging Between Users
- Public User Profiles
- Saved Items (Bookmark)
- Success Stories Sharing
- Urgent Lost Items Feed
- GPS Based "Near Me" Search
- Interactive Map & Heatmap
- Police FIR PDF Generator
- QR Code Generator for Items
- Reputation & Badge System
- Admin Dashboard with Analytics
- CSV Data Export
- Mobile Responsive Interface

## ▶️ How to Run the Project

1. Clone the repository

```
git clone https://github.com/YOUR_USERNAME/intelligent-lost-found-management-system.git
```

2. Navigate into the folder

```
cd intelligent-lost-found-management-system
```

3. Install dependencies

```
pip install django pillow
```

4. Run the server

```
python manage.py runserver
```

5. Open browser

```
http://127.0.0.1:8000
```
